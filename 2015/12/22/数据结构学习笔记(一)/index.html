数据结构学习笔记(一)

数据的逻辑结构

根据数据元素关系间的基本特性，有四种基本数据结构：

集合（数据元素同属一个集合，但无其他关系）
线性结构（一对一）
树形结构（一对多）
图状结构（多对多）

数据的物理结构

亦称存储结构，是数据在计算机存储器内的表示。存储结构可分为4大类:

顺序（借助元素在存储器中的相对位置来表示）
链式（借助指示元素存储地址的指针来表示）
索引
散列

数据的运算

常用的数据运算方法有5种:

插入,删除,修改,查找,排序

算法特性

输入,输出,有穷性,确定性,可行性

算法效率的度量

O(1) < O(logn) < O(n) < O(nlogn) < O(n²) < O(n³) < O(2的n次方) < O(n!) < O(n的n次方)
